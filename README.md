## Docs

프로젝트 세팅

1. 패키지 설치

```
yarn install
```

2. `.env` 파일 세팅

`.env` 파일을 만든 후 아래 firebase 설정값을 세팅한다.

```
REACT_APP_FIREBASE_API_KEY=
REACT_APP_FIREBASE_MESSAGE_SENDER_ID=
REACT_APP_FIREBASE_APP_ID=
REACT_APP_FIREBASE_MEASUREMENT_ID=
```
